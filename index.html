getNya

Zerochan.net pictures with preview

Download this project as a .zip file Download this project as a tar.gz file

Simple java application for downloading pictures from Zerochan.net. You can find images by size or a tag. It's simple. And flat.

All you need to do: download .jar file and run it with Oracle JVM (or any another JVM supporting image decoding). Let's nya!
